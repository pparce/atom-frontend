# Atom frontend

Aplicación web desarrollada como parte del challenge técnico Fullstack.

## Tecnologías

- **Node.js****
- **TypeScript**
- **Angular v19**
- **Firebase Hosting**
- **Bootstrap v5**
- **FontAwesome**

---

## Instalación

1. Cloná el repositorio

```bash
git clone https://github.com/pparce/atom-frontend.git
cd atom-frontend
npm install
```

## ⚙️ Scripts disponibles

```bash
npm run start
npm run build
npm run deploy
```
